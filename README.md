# Google Play Store Data EDA

## Project Overview
Exploratory Data Analysis (EDA) on Google Play Store data using Python.

## Key Steps
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Visual insights on ratings, installs, categories, and content ratings
- Correlation analysis between numerical features

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Dataset
Google Play Store dataset
